# rust-simple-gossip

A simple example of a gossip protocol utilizing Rust.

## Requirements

- Rust 1.75.0 or later
- Cargo

## Installation

Clone the repository:

```bash
$ git clone https://github.com/Kourin1996/rust-gossip-app.git
$ cd rust-gossip-app

$ cargo build
```

## Usage

```bash
$ ./build.sh
$ ./peer --port 8080 --period 5
```

## Test

```bash
$ cargo test
```
